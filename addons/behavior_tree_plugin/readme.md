# 适用于Godot的 行为树编辑器插件
将本仓库克隆到`res://addons`文件夹下面，然后在Godot编辑器中启用本插件即可。

# 特性
- 添加一个行为树节点编辑器，方便创建行为树。
- 添加了行为树节点和若干行为树节点的节点。
- 提供脚本模板，用于方便地创建自定义的动作脚本和条件。将script_templates文件夹复制覆盖到项目资源根目录即可使用。
