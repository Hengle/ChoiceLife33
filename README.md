# 武侠与江湖 饭店打工小游戏

## 参加群内 game jam 活动，做的小游戏。

### 说明：
1. 只包含2个小游戏，且完成度还很低。
2. 做菜游戏，就是一个简单的猜配方游戏，根据菜名，猜配方，选择正确加分。
3. 算账游戏，就是一个算数游戏，只是需要在虚拟的算盘上打出正确的数字（也许会点珠算口诀会快一点。~ ~）
4. 计划作为《武侠与江湖》 里面其中的俩个小游戏。
